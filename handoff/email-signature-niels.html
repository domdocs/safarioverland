How to install this in Mac Mail

  1. Open Mail → Settings → Signatures
  2. Click + under the middle column to create a new signature (any account; you can copy to "All Signatures" after)
  3. Give it a name, e.g. Niels — Safari Overland
  4. Leave the placeholder text in the right pane for now
  5. Come back here, click anywhere inside the dashed box below, press ⌘A to select all, then ⌘C to copy
  6. Switch back to Mail's signature pane, click into the right pane, ⌘A to select the placeholder text, then ⌘V to paste
  7. Make sure "Always match my default message font" is un-ticked
  8. Close Settings — signature is saved
  9. Set this signature as Niels' default for new messages on the niels@safarioverland.com account
